Toolbar Design — Structure & Modes

How the HTML is structured and how each mode behaves. Resize your browser to see responsive behavior.
Section 1 — HTML Structure
Core layout: two sections in a wrapping flex row
On desktop, start expands and end shrinks. When start is too wide, end wraps below. On mobile, they stack.
<div class="d-flex flex-column flex-sm-row flex-wrap align-items-stretch align-items-sm-center gap-2">
  <div class="flex-grow-1" style="min-width:0"> start — title, breadcrumbs, etc. </div>
  <div class="d-flex align-items-center gap-2 flex-wrap OR flex-nowrap overflow-x-auto"> end — actions </div>
</div>
Mobile stack order — swap flex-column → flex-column-reverse to put end first on mobile.
