{% load flex_menu %} {% comment %} c-app.sidebar ============= Root application sidebar component. Renders an AdminLTE-compatible
element containing a sticky brand header, a scrollable navigation area, a sticky footer strip, and a full-screen overlay for dismissing the sidebar on mobile. Attributes ---------- fixed (bool) Whether the sidebar is constrained to the viewport height. collapsed (bool) Whether the sidebar starts in a collapsed state collapsible (bool) Whether the sidebar collapses to icons only expand (str) The breakpoint at which the sidebar is automatically expanded expand-on-hover (bool) When in "mini" mode, automatically expands the sidebar when the user hovers. indent (bool) Indents nested nav items. compact (bool) Applies less spacing between nav elements. dark (bool) Makes the sidebar ALWAYS use the dark theme. brand-text (str) Display name shown in the brand header. brand-url (str) The url the user is directed to when click the sidebar header class (str) Extra CSS classes appended to the
. menu (str) Name of a flex-menu Menu class to render. Default: "AppMenu". {% endcomment %}
